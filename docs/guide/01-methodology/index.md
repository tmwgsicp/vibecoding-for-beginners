## 为什么方法论最重要？

**方法论 ≠ 教你写代码**

方法论教的是:
- 如何思考问题
- 如何做决策
- 如何与AI协作
- 如何解决问题

```
没有方法论:
遇到问题 → 不知道怎么办 → Google乱搜 → 试了很多方法 → 还是不行 → 放弃

有方法论:
遇到问题 → 分析问题类型 → 查对应章节 → 找到解决思路 → 快速解决 → 继续前进
```

**简单说:** 方法论是做产品的地图。

- 问题发现（1-3章）

### 建议2: 边做边查
不需要背下来。做项目时:
- 不知道怎么开始 → 看第1-3章
- 不知道选什么技术 → 看第4-5章
- 不知道怎么和AI对话 → 看第7章

**方法论是工具书，不是教科书。**
