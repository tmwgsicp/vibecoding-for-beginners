## 🎯 3 个立即行动

### 行动 1: 测试写作助手(5 分钟)

**在 Cursor 中输入:**
```
我要写"如何与 AI 协作"这一章,我先说我的核心经验:

很多人用 AI 开发就像用 Google 翻译,输入中文得到英文,
完全不管中间发生了什么。

这样很危险:
1. 不知道 AI 为什么这么写
2. 出 bug 了不知道怎么改
3. 完全依赖 AI,自己什么都不会

我的做法是先让 AI 解释方案,我听懂了再让它实现...
```

**期望:**
- Skill 自动激活
- 输出结构化教程
- 保持你的原话和观点
- 标注内容来源

**用途:** 立即用来写课程内容!

### 行动 2: 测试 Skill Creator(10 分钟) ⭐

```
我想创建一个 Skill,每次写 README 时能有统一格式,
包含项目介绍、安装、使用这些部分
```

**期望:**

**用途:** 10 分钟创建你的第一个 Skill!
